# Development Guide Setp-by-Step Instructions

This guide was structured aiming on define the steps I should have used to create this project at first.

## Project Objective

> First of all, before creating a project we should make clear what is its purpose and which technologies we are going to use.

- **Objective:** Create a simple full-stack application with emphasis on the back-end services. This project is a dashboard with autentication and RBAC (Role Based Access Control). It should be strongly secure and performatic.
- **Stack:**
- - `Node.js`/`TypeScript`
- - `Vite` + `TailwindCSS` + `Shadcn/ui` | Front-end
- - `Fastify` | Back-end
- - `Prisma ORM` + `PostgreSQL` | Database
- - `JWT` + `Bcrypt` | Security and Hashing
